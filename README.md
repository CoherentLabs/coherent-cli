# Usage
<!-- usage -->
```sh-session
$ npm install -g coherent-cli
$ coherent-cli COMMAND
running command...
$ coherent-cli (-v|--version|version)
coherent-cli/0.0.0 win32-x64 node-v12.19.0
$ coherent-cli --help [COMMAND]
USAGE
  $ coherent-cli COMMAND
...
```
